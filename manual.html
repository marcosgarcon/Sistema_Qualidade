Manual do Usuário

Visão Geral do Sistema

Bem-vindo ao Pan Eng Process & Quality. Este é um sistema completo de ferramentas de qualidade que funciona diretamente no seu navegador, sem necessidade de internet ou instalação. Todos os dados que você insere são salvos no seu próprio computador.

Funcionalidades Globais

Existem duas funções na base da barra lateral que são essenciais para a segurança dos seus dados:

  • Backup: Ao clicar neste botão, o sistema gera um único arquivo .json que contém todos os dados de todas as ferramentas. Salve este arquivo regularmente em um local seguro (seu computador, um pendrive, na nuvem).
  • Restaurar: Se você trocar de computador, limpar o cache do navegador ou precisar recuperar seus dados, use este botão. Ele pedirá para você selecionar o arquivo de backup .json que você salvou, e irá restaurar todo o sistema para aquele ponto. Atenção: a restauração apaga os dados atuais para substituí-los pelos do backup.

Guia de Ferramentas

As ferramentas são divididas em categorias. Abaixo está uma explicação de como usar os principais tipos de ferramentas.

Ferramentas de Formulário (5 Porquês, SWOT, A3, etc.)

Estas ferramentas são projetadas para a coleta estruturada de informações.

  1. Navegue até a ferramenta desejada na barra lateral.
  2. Preencha os campos de texto com as informações da sua análise.
  3. Clique no botão "Salvar" no final da página.
  4. Seus dados ficam salvos. Ao reabrir a página, eles serão carregados automaticamente. Use o botão "Limpar" para começar um novo formulário.

Ferramentas com Gráficos (Pareto, Histograma, Dispersão)

Estas ferramentas transformam dados brutos em análises visuais.

  1. Navegue até a ferramenta (ex: Diagrama de Pareto).
  2. Na tabela "Dados de Entrada", preencha as informações (ex: causas e frequências). Use o botão "Adicionar Linha" se precisar de mais campos.
  3. Clique no botão "Gerar/Atualizar Gráfico". O gráfico será desenhado abaixo.
  4. Os dados da tabela são salvos automaticamente. Para compartilhar, use o botão "Exportar para CSV", que gera um arquivo compatível com o Excel.

Guia Detalhado: Cronoanálise MTM por Vídeo

Esta é a ferramenta mais complexa e poderosa. Siga estes passos:

  1. Carregue um Vídeo: Clique em "Carregar vídeo" e selecione um arquivo de operação do seu computador. O vídeo não sai da sua máquina.
  2. Marque as Etapas: Dê play no vídeo. Ao final de cada movimento ou etapa que você quer medir, clique no botão "Marcar Etapa". Uma nova linha aparecerá na "Tabela de Análise MTM".
  3. Descreva e Codifique: Na nova linha que apareceu, você pode digitar uma descrição para a etapa. Se você usa a metodologia MTM, pode inserir um código (ex: `G1A`) no campo "Código MTM".
  4. Use a Base MTM: Na parte inferior da página, há uma "Base de Dados MTM" editável. Se o código que você digitou na tabela de análise (passo 3) existir nesta base, o sistema preencherá automaticamente os campos "Tipo" e "TMU". Você pode adicionar, editar ou remover códigos nesta tabela base conforme sua necessidade.
  5. Analise e Salve: Os tempos e totais são calculados automaticamente. Use a seção "Análise e Oportunidades de Melhoria" para suas conclusões e clique em "Salvar Análise".

Dicas Importantes

  • FAÇA BACKUPS REGULARMENTE! A função de backup é sua maior garantia de que não perderá seu trabalho.
  • Os dados são salvos por navegador. Se você usar o Chrome e depois o Firefox, os dados não serão compartilhados entre eles.
  • Limpar o "cache" ou os "dados de navegação" do seu navegador irá **APAGAR TODOS OS DADOS** do sistema. Tenha sempre um backup antes de fazer isso.
